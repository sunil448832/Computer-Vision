This folder contain two progarm file main.py and util.py.
To run the progarm execute the following command in terminal.
1. with overlapping 
   python main.py --block_size 8 16 32
2. without overlapping 
   python main.py --block_size 8 16 32 --overlapping 'no'

When the progarm get executed one folder named "images" will be created,
and the reconstructed image will stored in that folder.
